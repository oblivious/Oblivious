Release Notes - CruiseControl .NET - Version 1.0.1

This release fixes a number of key bugs in the 1.0 release.

Bug

  • CCNET-606 - ModificationWriterTask should write xml as UTF8
  • CCNET-608 - Cannot download cctray from installed webdashboard virtual directory
  • CCNET-609 - Starteam "Auto Get Latest" gets source files with old timestamps
  • CCNET-620 - Compile-msbuild stylesheet has invalid element: $warning.events
  • CCNET-623 - Cvs reports modified files as added
  • CCNET-624 - Error installing multiple instances of CCService
  • CCNET-628 - State files should use UTF-8 xml declaration
  • CCNET-629 - Project Trigger: Default Url is incorrect – contains a {0}
  • CCNET-631 - VSS provider should use current culture to parse and output dates
  • CCNET-636 - ProjectTrigger: Leaving "serverUri" does not work
  • CCNET-643 - UrlTrigger hammers web server

Improvement

  • CCNET\-645 - CCNet should add <supportruntime> for 2.0
