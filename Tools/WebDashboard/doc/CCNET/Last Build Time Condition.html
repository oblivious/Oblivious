Last Build Time Condition

Checks if the last build started at least a certain time period ago.

Version

Available from version 1.6

Examples

Basic example
<lastBuildTimeCondition time="120" />
Example in context
<conditional>
  <conditions>
    <lastBuildTimeCondition>
      <time unit="minutes">5</time>
    </lastBuildTimeCondition>
  </conditions>
  <tasks>
    <!-- Tasks to perform if condition passed -->
  </tasks>
  <elseTasks>
    <!-- Tasks to perform if condition failed -->
  </elseTasks>
</conditional>

Configuration Elements

Element Description Type Required Default Version
description A description of the condition. String No none 1.6
time The time period to use. Timeout Configuration Yes n/a 1.6
