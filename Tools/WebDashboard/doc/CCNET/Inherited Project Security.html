Inherited Project Security

Inherit the security settings for a project from the server settings.

Version

Available from version 1.5

Examples

<security type="inheritedProjectSecurity" />

Configuration Elements

Element Description Type Required Default Version
type The type of the project security. String - must be inheritedProjectSecurity Yes n/a 1.5
guest The name of the account to use for guests. String No None 1.5
