Server Quick Start

  1. Check you have the Installation Prerequisites.
  2. Download and Install. The Basic Installation page describes the types of kits that are available.
  3. Add a Project Configuration Block to ccnet.config with the following sub blocks (you'll want at least these for most usages of CruiseControl.NET):
    • SourceControl block to specify where the project will be checked out from, see Source Control Blocks.
    • Tasks block to specify the steps required for the build, see Task And Publisher Blocks.
    • Publishers block to define generated output, see Task And Publisher Blocks.
  4. Start the server running either:
    • From the command line, see The Server Console Application
    • As a windows service, See The Server Service Application.
