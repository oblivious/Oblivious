Highlights of the CCNet 1.4.4 RC2 release

This is the second Release Candidate for CruiseControl.NET 1.4.4. It incorporates a number of bug fixes and enhancements based on the feedback that we have received after releasing the first release candidate.
We welcome futher feedback on this candidate to improve the final distribution of 1.4.4.

Upgrading to CCNet 1.4.4 RC2

There are no breaking changes for people coming from 1.4.4 RC1

Upgrading from a version before 1.4.4 RC1 ?

See the release notes for the CCNET 1.4.4 RC1 Release Notes.
