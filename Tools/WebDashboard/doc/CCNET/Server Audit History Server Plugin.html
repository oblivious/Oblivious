Server Audit History Server Plugin

The Server Audit History Server Plugin displays the audit log from the server. server is running.

Version

Available from version 1.5

Examples

<serverAuditHistoryServerPlugin />

Configuration Elements

There is no configuration for this plugin.

Notes

This requires that the currently logged in user has the required permissions on the server.
