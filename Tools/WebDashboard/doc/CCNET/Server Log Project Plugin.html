Server Log Project Plugin

The Server Log Project Plugin shows you recent activity that has been output to the server log for a specific project.

Version

Available from version 1.0

Examples

<serverLogProjectPlugin />

Configuration Elements

There is no configuration for this plugin.

Notes

Read the Server Application Config File page for more help on build server logging.
