CruiseControl.NET : projectConfigurationServerPlugin
This page last changed on Oct 30, 2008 by williams.

This plugin shows the basic configuration of the projects on this buildserver.
Like project name, category, queue and queue priority, making it easier to spot conflicts in the queue setup for this buildserver.

Example

<projectConfigurationServerPlugin/>

Configuration Elements

There is no configuration for this plugin.
