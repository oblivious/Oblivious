Project Report Project Plugin

The Project Report Project Plugin shows you summary details for a specific project. Part of these details are any External Links you have specified in the project configuration.

Version

Available from version 1.0

Examples

<projectReportProjectPlugin />

Configuration Elements

There is no configuration for this plugin.
